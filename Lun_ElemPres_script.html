SAMPLE CODE A

 
<script>

	function colorChange() {

		let f_color = document.getElementById("f-page-color").value;

		if (f_color == "red") {
			document.documentElement.style.setProperty('--primary-color', '#D42A2A');
		}

		if (f_color == "blue") {
			document.documentElement.style.setProperty('--primary-color', '#2AD4D4');
		}

		if (f_color == "green") {
			document.documentElement.style.setProperty('--primary-color', '#84d728');
		}

		if (f_color == "yellow") {
			document.documentElement.style.setProperty('--primary-color', '#e9c916');
		}
	}

</script>

ASSOCIATED
ELEMENTS

<noscript>

Defines alternate content to display if scripts are disabled.

ATTRIBUTES

Attribute

Value

Description

async

async

Specifies that the script is to be run in parralel while the HTML parses for external scripts.

charset

charset

Defines the character encoding that is being used in an external script

defer

defer

Specifies that the script is to be run in series with the HTML for external scripts; after the HTML parses.

src

URL

Specifies the location of an external script.

type

media_type

Specifies what type of script is being referred to.

GLOBAL ATTRIBUTES

No global attributes are useful for script elements as they are by default hidden.

EVENT ATTRIBUTES

Event attributes are not supported for the script element.

Events call the scripts. Not the other way around.

PROPERTIES

Property

Value

Description

display

none

Scripts are hidden from view.
