ASSOCIATED
ELEMENTS

<input>

Creates an input field to enter data into.

<textarea>

Creates a multi-line text input field.

<button>

Creates a button that can accept nested elements different from <input> buttons which cannot.

<select>

Creates a drop-down list that collects user input.

<option>

Defines the options within a drop-down list created by <select>

<optgroup>

Groups options together within a drop-down list.

<fieldset>

Groups specified elements together in a form with a box.

<label>

Creates a label to be used with different input types such as <textarea> or radio buttons.

<output>

Represents the result of a calculation between different inputs.

ATTRIBUTES

Attribute

Value

Description

accept-charset

character_set

Defines the character encoding that is to be used for form submission.

action

URL

Defines the location to send the form's data once it's been submitted.

autocomplete

on, off

Turns autocomplete on or off.

enctype

application/x-www-form-urlencoded,
multipart/form-data, text/plain

Defines the data's encryption format when submitting to a server.

method

get, post

Defines the HTTP method to use to send data.

name

text

Defines a form's name

novalidate

novalidate

Flags that a form should not be validated upon submission.

rel

external, license,
next, nofollow,
noopener, noreferrer,
opener, prev, search

Defines the relationship between the linked and current document.

target

_blank, _self, _parent, _top

Defines the place to display the response after the form has been submitted.

GLOBAL ATTRIBUTES

spellcheck

true, false

Specifies whether on not spellcheck should be applied. Useful for text inputs.

class

classname

Specifies a CSS class that can be used to style the form

EVENT ATTRIBUTES

onchange

script

Calls the specified script when any element within the form is changed.

onsubmit

script

Calls the specified script when the form is submitted.

PROPERTIES

Property

Value

Description

display

block

Forms are displayed as blocks of content, the same as a div element.

margin-top

0em

Forms do not have a top margin.
